安裝 Nginx 1.9+

Thu, 05 May 2016 02:47:01 +0800

Contents

    • 新增nginx.repo
    • 安裝Nginx
    • 啟動服務

新增nginx.repo

# vi /etc/yum.repos.d/nginx.repo

[nginx]
name=nginx repo
baseurl=http://nginx.org/packages/mainline/centos/$releasever/$basearch/
gpgcheck=0
enabled=1

安裝Nginx

# yum install nginx

啟動服務

# service nginx start

# chkconfig --levels 235 nginx on
